# PWA Иконки

Для полной поддержки PWA необходимо создать иконки следующих размеров:

## Необходимые размеры:
- ✅ `favicon-16x16.png` - есть
- ✅ `favicon-32x32.png` - есть  
- ✅ `apple-touch-icon.png` (180x180) - есть
- ❌ `icon-192x192.png` - нужно создать
- ❌ `icon-512x512.png` - нужно создать

## Как создать иконки:

1. Используйте существующий SVG логотип: `public/logofiol.svg`
2. Экспортируйте PNG в нужных размерах (192x192 и 512x512)
3. Поместите файлы в папку `public/`

Или используйте онлайн-инструменты:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Временное решение:

До создания правильных иконок, можно использовать существующий `apple-touch-icon.png`:
- Скопировать в `icon-192x192.png` (будет немного растянуто, но работает)
- Масштабировать до `icon-512x512.png`

ВНИМАНИЕ: Для продакшена рекомендуется создать качественные иконки из SVG в точных размерах.
